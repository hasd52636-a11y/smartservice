# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Initial release of AI Virtual Customer Service System
- Core features:
  - Project management system
  - Intelligent knowledge base
  - AI conversation system with multi-modal support
  - Ticket management system
  - Data analytics platform
  - Video chat capabilities
  - Smart recommendation system

### Fixed
- Security improvements:
  - API key encryption storage
  - CORS configuration hardening
- Code quality:
  - Route configuration optimization
  - TypeScript configuration improvements
  - ESLint and Prettier integration
- Performance:
  - Storage management
  - Timer cleanup
  - Logging system

### Changed
- Architecture improvements
- Component organization
- Service layer optimization

## [0.9.0] - 2023-12-01

### Added
- Beta features for testing
- Knowledge graph prototype
- Internationalization support (zh-CN, en-US, ja-JP, ko-KR)

### Known Issues
- Some features are using mock data (marked with "simulated" badges)
- Full video generation not yet implemented
- Knowledge graph is tag-based only
