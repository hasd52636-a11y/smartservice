# 🔑 API密钥配置快速指南

## 问题解决

如果遇到以下错误：
- `/api/zhipu/embeddings` 返回 500 错误
- 知识库向量化失败
- AI功能无法正常使用

## 解决步骤

### 1. 获取API密钥
1. 访问 [智谱AI开放平台](https://open.bigmodel.cn/)
2. 注册/登录账号
3. 在控制台创建API密钥

### 2. 配置API密钥
**方法A：管理后台配置（推荐）**
1. 进入项目管理页面
2. 选择任意项目
3. 点击"4. 客服回复设置"标签
4. 在"API 密钥配置"部分输入密钥
5. 点击"测试连接"验证

**方法B：调试工具**
1. 访问 `debug-api-setup.html`
2. 按照页面指引配置

### 3. 生产环境配置
在Vercel Dashboard设置环境变量：
- Name: `ZHIPU_API_KEY`
- Value: 你的API密钥

## 验证配置
1. 扫描二维码进入用户对话页面
2. 发送消息测试AI回复
3. 检查控制台是否还有错误
