# 智谱AI API 配置指南

## 获取API密钥

1. 访问 [智谱AI开放平台](https://bigmodel.cn)
2. 注册并登录账户
3. 进入 [API密钥管理页面](https://bigmodel.cn/usercenter/proj-mgmt/apikeys)
4. 点击"创建API密钥"
5. 复制生成的API密钥

## 配置API密钥

### 方法1: 在项目设置中配置（推荐）
1. 打开项目详情页面
2. 点击"AI配置"标签
3. 在"API密钥配置"部分输入您的智谱AI API密钥
4. 点击"测试连接"验证配置是否正确

### 方法2: 使用测试工具
1. 打开 `test-zhipu-api.html` 文件
2. 在浏览器中访问该文件
3. 输入API密钥并测试连接

## API端点信息

- **基础URL**: `https://open.bigmodel.cn/api/paas/v4`
- **认证方式**: Bearer Token
- **支持的模型**:
  - `glm-4.7` - 最新旗舰模型
  - `glm-4.6v` - 视觉模型（支持图片分析）
  - `glm-4-voice` - 语音模型
  - 更多模型请参考官方文档

## 常见问题

### 404错误
- 检查API端点是否正确
- 确认网络连接正常
- 验证代理配置是否正确

### 401错误
- 检查API密钥是否正确
- 确认API密钥是否已激活
- 检查账户余额是否充足

### 429错误
- 请求频率过高，请降低请求频率
- 检查并发数是否超限
- 联系客服增加限额

## 技术支持

如遇到问题，请：
1. 检查浏览器控制台的错误信息
2. 使用测试工具验证API连接
3. 参考 [智谱AI官方文档](https://docs.bigmodel.cn)
4. 联系技术支持

## 安全提醒

- 请妥善保管您的API密钥
- 不要在代码中硬编码API密钥
- 定期更换API密钥以确保安全
- API密钥仅保存在本地浏览器中，不会上传到服务器
