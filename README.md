# SmartGuide AI - 智能产品客服系统

一个基于智谱AI的智能产品客服系统，支持扫码访问、多模态交互、知识库管理等功能。

## 🚀 功能特性

### 用户端功能
- **扫码访问** - 用户扫描产品二维码即可获得智能客服支持
- **多模态交互** - 支持文字、语音、图片、视频等多种交互方式
- **智能问答** - 基于产品知识库的精准问答
- **实时对话** - 流式响应，自然对话体验
- **OCR识别** - 图片文字识别功能
- **视频客服** - 实时视频指导和标注功能

### 商家端功能
- **项目管理** - 创建和管理多个产品项目
- **知识库管理** - 维护产品知识库，支持向量化搜索
- **二维码生成** - 为每个产品生成专属二维码
- **数据分析** - 用户访问和使用情况统计
- **系统诊断** - 实时监控系统运行状态
- **API配置** - 灵活的AI服务配置

## 🛠️ 技术栈

- **前端**: React 19 + TypeScript + Tailwind CSS
- **路由**: React Router DOM
- **构建工具**: Vite
- **AI服务**: 智谱AI (GLM-4.7, GLM-4.6v, Embedding-3)
- **部署**: Vercel
- **图标**: Lucide React
- **图表**: Recharts

## 📦 部署指南

### 1. 环境要求
- Node.js 24.x
- npm 或 yarn

### 2. 环境变量配置

在 Vercel Dashboard 中配置以下环境变量：

```bash
# 智谱AI API密钥（必需）
ZHIPU_API_KEY=your_zhipu_api_key_here

# 应用配置
VITE_APP_TITLE=SmartGuide AI
VITE_APP_VERSION=1.0.0
VITE_ZHIPU_BASE_URL=https://open.bigmodel.cn/api/paas/v4
```

### 3. 部署到 Vercel

1. 将代码推送到 GitHub 仓库
2. 在 Vercel 中导入项目
3. 配置环境变量
4. 部署完成

### 4. 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 🔧 配置说明

### API密钥获取
1. 访问 [智谱AI开放平台](https://open.bigmodel.cn/)
2. 注册账号并获取API密钥
3. 在Vercel环境变量中配置 `ZHIPU_API_KEY`

### 域名配置
系统支持自动域名检测，生产环境建议使用自定义域名。

## 📱 使用流程

### 商家端
1. 访问管理后台 `/admin`
2. 创建产品项目
3. 配置AI服务参数
4. 添加产品知识库
5. 生成产品二维码
6. 将二维码印刷到产品包装

### 用户端
1. 扫描产品二维码
2. 进入智能客服界面
3. 通过文字、语音或图片提问
4. 获得AI智能回答

## 🐛 故障排查

### 常见问题
1. **API密钥错误** - 检查Vercel环境变量配置
2. **二维码无效** - 确认项目ID正确且项目状态正常
3. **AI回答不准确** - 检查知识库内容是否完整
4. **语音功能异常** - 确认浏览器麦克风权限

### 系统诊断
访问 `/admin/diagnostics` 进行系统健康检查
